# Mailchimp Integration

This guide walks you through setting up the Mailchimp integration within Boltic.

## Prerequisites

To set up the Mailchimp integration, you'll need:

- Mailchimp API key or OAuth2 credentials (client ID and client secret)
- Data center prefix from your API key
- Admin access to the Mailchimp account

## Setting Up the Mailchimp Integration in Boltic

### 1. Log In

Log into your Boltic account or navigate to the Boltic Open Source dashboard.

### 2. Navigate to Integrations from the Primary Menu

Click on **Sources** and then click on **+ Add New source**.

### 3. Select Source Type

On the **Set up the source** page, search for Mailchimp from the search bar and select it.

### 4. Configure Integration

#### API Key Authentication

- **Authentication Type:** Select: `API Key`
- **API Key:** Enter: Your Mailchimp API Key. Refer to the Mailchimp API authentication documentation for detailed instructions on getting your API key.
- **Data Center:** Enter: The server prefix from your API key (e.g., us1).

#### OAuth2 Authentication (Currently commented out, uncomment to use)

- **Authentication Type:** Select: `OAuth2`
- **Client ID:** Enter: Your Mailchimp app client ID.
- **Client Secret:** Enter: Your Mailchimp app client secret.
- **Redirect URI:** Enter: `v1/mailchimp/oauth2/callback`.
- **Connect Mailchimp Account:** Button: Click to connect your Mailchimp account. Follow the on-screen instructions to complete the OAuth2 flow.
- **Mailchimp Account Connected:** Button: Once connected, this will confirm your Mailchimp account is linked.

### 5. Test and Save Integration

Click **Test Connection** to ensure the provided details are correct and the connection is successful. Once the connection test is successful, click **Set up source** to complete the configuration.

## Troubleshooting

If you encounter any issues during setup:

- **Check API Key or OAuth2 credentials:** Ensure the provided details are correct and have the necessary permissions.
- **Verify Data Center Prefix:** Confirm the data center prefix from your API key is correct.
- **Network Issues:** Check for any network issues that might be affecting connectivity.

## Frequently Asked Questions (FAQs)

1. **How do I find my Mailchimp API key?**

   To find your Mailchimp API key, navigate to your Mailchimp account settings and look for the API key section. You can also refer to Mailchimp's official documentation for detailed instructions.

2. **What should I do if my connection fails?**

   Ensure your API key or OAuth2 credentials are correct.
   Verify the data center prefix and ensure there are no typos.
   Check your network connection and firewall settings.

3. **How often is the data synced?**

   The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.

4. **Can I customize the data being synced?**

   Yes, you can customize the data being synced by adjusting the settings during the setup process.
