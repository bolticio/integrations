# PixelBin

Use the PixelBin activity to automate work in PixelBin, and integrate PixelBin with other activities. Boltic has built-in support for a wide range of PixelBin features, including uploading file using URL source, updating uploaded file, deleting the uploaded file, and Transforming the images.

## PixelBin Secrets

### Prerequisites

-   Create a PixelBin account.

### Supported Authentication Methods

-   API key

### Related Resources

Refer to [PixelBin documentation](https://www.pixelbin.io/docs/) for more information about the different features and services.

### Using API Key

To configure this secret, you'll need:

1. **API token**: Refer to the [Create PixelBin API token documentation](https://www.pixelbin.io/docs/tokens/create-token/) for detailed instructions on getting your API key.

## Basic Operations

### File

-   Upload a file using public source url
-   Update a uploaded file
-   Delete a uploaded file

-   [Basic Image Transformations](https://www.pixelbin.io/docs/transformations/basic/)
