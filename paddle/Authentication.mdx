"# Paddle\n\n## Overview\n\nPaddle is a complete payments and billing platform for SaaS companies. It provides a comprehensive solution for handling subscriptions, one-time payments, tax compliance, and billing operations. \n\nYou can access Paddle's functionality programmatically using the Paddle API.\n\n## Authentication Ways\n\n- Personal API Key\n\n## Personal API Key\n\nTo authenticate with Paddle, you need to create a Personal API Key in your Paddle dashboard. This key provides access to your Paddle account data and allows you to perform operations through the API.\n\n**Steps:**\n\n1. **Create a Paddle Account**\n    - Go to <a href=\"https://login.paddle.com/signup\" target=\"_blank\" rel=\"noopener\"><span>Paddle sign-up page</span></a>\n    - Create your account and complete the setup process\n    - Verify your email address and complete any required verification steps\n\n2. **Access Developer Tools**\n    - Log into your Paddle dashboard\n    - Navigate to **Developer Tools** in the main menu\n    - Click on the **Authentication** tab\n\n3. **Generate Your API Key**\n    - In the Authentication section, locate the **API keys** area\n    - You can either:\n        - Copy an existing key if one is already created\n        - Click **Create New Key** to generate a new Personal API Key\n    - Ensure that provide Notification(Read) access is provided to the API Key. This is used to test the authentication flow.\n    - Store this key securely as it will be used for authentication\n\n4. **Select Environment**\n    - Choose the appropriate environment for your integration:\n        - **Sandbox**: For testing and development\n        - **Live**: For production use\n    - Ensure you're using the correct environment URL:\n        - Sandbox: `https://sandbox-api.paddle.com`\n        - Live: `https://api.paddle.com`\n\nFor more details, see the <a href=\"https://developer.paddle.com/api-reference/about/authentication\" target=\"_blank\" rel=\"noopener\"><span>Paddle API Authentication documentation</span></a>.\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io\" target=\"_blank\" rel=\"noopener\"><span>Support Documentation</span></a>.\n"
