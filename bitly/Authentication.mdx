"  \n\n# Bitly Integration\n\n  \n\nThis guide walks you through setting up the Bitly integration within Boltic.\n\n  \n\n## Prerequisites\n\nTo set up the Bitly integration, you'll need:\n\n- The Bitly personal access token\n\n- Admin access to the Bitly account\n\n  \n\n## Setting Up the Bitly Integration in Boltic\n\n  \n\n1.  **Log In**\n\n\t- Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n  \n\n2.  **Navigate to Integrations from the Primary Menu**\n\n\t- Click on `Sources` and then click on `+ Add New source`.\n\n  \n\n3.  **Select Source Type**\n\n\t- On the Set up the source page, search for `Bitly` from the search bar and select it.\n\n  \n\n4.  **Configure Integration**\n\n\t-  **Name:** Enter a name for the Bitly integration.\n\n\t-  **Auth type:** Select Authentication type to Personal Access Token.\n\n\t-  **Personal Access Token:** Enter your Bitly personal access token. The Bitly integration currently supports only the \"Access token\" authentication type.\n\n  \n\n5.  **Details (Optional)**\n\n\t-  **Tags:** Add any tags for categorization.\n\n\t-  **Description:** Provide a brief description of the integration.\n\n  \n\n6.  **Test and Save Integration**\n\n\t- Click `Test Connection` to ensure the provided details are correct and the connection is successful.\n\n\t- Once the connection test is successful, click `Set up source` to complete the configuration.\n\n  \n\n## Troubleshooting\n\nIf you encounter any issues during setup:\n\n-  **Check Access Token:** Ensure the provided personal access token is correct and has the necessary permissions.\n\n-  **Network Issues:** Check for any network issues that might be affecting connectivity.\n\n  \n\n## Frequently Asked Questions (FAQs)\n\n1.  **How do I find my Bitly personal access token?**\n\n- To find your Bitly personal access token, navigate to your Bitly account settings and look for the API token section. You can also refer to [Bitly's official documentation](https://dev.bitly.com/docs/getting-started/authentication/) for detailed instructions.\n\n\n2.  **What should I do if my connection fails?**\n\n- Ensure your personal access token is correct.\n\n- Check your network connection and firewall settings.\n\n  \n\n3.  **How often is the data synced?**\n\n- The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.\n\n  \n\n4.  **Can I customize the data being synced?**\n\n- Yes, you can customize the data being synced by adjusting the settings during the setup process.\n\n  \n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our [Support Documentation](https://docs.boltic.io)."
