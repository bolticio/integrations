

# Freshdesk



This guide walks you through setting up the Freshdesk integration within Boltic.



## Prerequisites

To set up the Freshdesk integration, you'll need:

- A Freshdesk account

- The Freshdesk domain name

- The Freshdesk API Key



## Setting Up the Freshdesk Integration in Boltic

1. **Log In**

   - Log into your Freshdesk account.



2. **Obtain API Key**

   - In the upper-right corner, click on your profile picture and select **Profile Settings**.

   - In the right sidebar, click the **View API key** button.

   - Copy the API key value shown and store it in a safe place.



3. **Log into Boltic**

   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.



4. **Navigate to Integrations**

   - Click on `Sources` and then click on `+ Add New source`.



5. **Select Source Type**

   - On the Set up the source page, search for `Freshdesk` from the search bar and select it.



6. **Configure Integration**

   - **Name:** Enter a name for the Freshdesk integration.

   - **Domain:** Enter your Freshdesk domain name. For example, if your Freshdesk URL is `https://abc.freshdesk.com`, enter `abc`.

   - **API Key:** Paste the API key copied from the Freshdesk Profile Settings.



7. **Details (Optional)**

   - **Tags:** Add any tags for categorization.

   - **Description:** Provide a brief description of the integration.



8. **Test and Save Integration**

   - Click `Test Connection` to ensure the provided details are correct and the connection is successful.

   - Once the connection test is successful, click `Set up source` to complete the configuration.



## Setting Up Freshdesk Webhooks

Freshdesk uses webhooks to trigger a scenario when an event occurs. To set up a webhook:

1. Add a Freshdesk instant module to your scenario and click **Create a webhook**.

2. **Webhook Name:** Enter a name for the webhook (optional).

3. **Connection:** Select the corresponding connection for the webhook.

4. Click **Save** and then **Copy address to clipboard**.

5. Log in to your Freshdesk account using <a href="https://support.freshdesk.com/support/login" target="_blank" rel="noopener"><span>Freshdesk Login</span></a>.

6. Follow the directions as outlined in the <a href="https://support.freshdesk.com/support/home" target="_blank" rel="noopener"><span>Freshdesk Using Webhooks in Automation documentation</span></a>.

Freshdesk will now send data to Boltic through the webhook when the selected event occurs.



## Frequently Asked Questions (FAQs)

1. **How do I find my Freshdesk API Key?**

   - Navigate to your Freshdesk account settings and look for the Profile Settings tab to find your API Key. Follow the instructions provided earlier in this guide.



2. **What should I do if my connection fails?**

   - Ensure your API Key is correct.

   - Verify the domain name and ensure there are no typos.

   - Check your network connection and firewall settings.



3. **How often is the data synced?**

   - The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.



4. **Can I customize the data being synced?**

   - Yes, you can customize the data being synced by adjusting the settings during the setup process.
