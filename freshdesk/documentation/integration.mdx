# Freshdesk

Use the Freshdesk activity to automate work in Freshdesk, and integrate Freshdesk with other activities. Boltic has built-in support for a wide range of Freshdesk features, including creating, updating, deleting, and retrieving contacts, tickets, and more.

## Freshdesk Secrets

### Prerequisites

- Create a Freshworks CRM account.

### Supported Authentication Methods

- API key

### Related Resources

Refer to [Freshworks CRM's API documentation](https://developers.freshworks.com/crm/api/) for more information about the service.

### Using API Key

To configure this secret, you'll need:

1. **API Key**: Refer to the [Freshdesk API authentication documentation](https://developers.freshdesk.com/api/#authentication) for detailed instructions on getting your API key.
2. **Your Freshdesk Domain**: Use the domain of your freshdesk account. This is part of the URL, for example, `https://<domain>.freshdesk.com`. So if you access Freshdesk through `https://boltic.freshdesk.com`, enter `boltic` as your Domain.

## Basic Operations

### Contact
- Create a contact
- Delete a contact
- Retrieve a contact
- Retrieve all contacts
- Update a contact

### Ticket
- Create a ticket
- Delete a ticket
- Retrieve a ticket
- Retrieve all tickets
- Update a ticket
