"This guide walks you through setting up the Jenkins integration within Boltic.\n\n#### Pre-requisites\n\nTo set up the Jenkins integration, you'll need:\n\n- The Jenkins domain URL\n\n- The Jenkins username\n\n- The Jenkins API token\n\n  \n\n#### Setting Up the Jenkins Integration in Boltic\n\n  \n\n1.  **Log In**\n\n\t- Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n  \n\n2.  **Navigate to Integrations from the Primary Menu**\n\n\t- Click on `Sources` and then click on `+ Add New source`.\n\t                          or\n\t- Click on `Destinations` and then click on `+ Add New destination`.\n\n  \n\n3.  **Select Source/Destination Type**\n\n\t- On the Set up the source/destination page, search for `JENKINS` from the search bar and select it.\n\n  \n\n4.  **Configure Integration**\n\n\t-  **Name:** Enter a name for the JENKINS integration.\n\n\t-  **Domain:** Enter your JENKINS domain URL.\n\n\t-  **Username:** Enter the username associated with your JENKINS account.\n\n\t-  **Personal API Token:** Enter your JENKINS personal api token. The JENKINS integration currently supports only the \"API Token\" authentication type.\n\n  \n\n5.  **Details (Optional)**\n\n\t-  **Tags:** Add any tags for categorization.\n\n\t-  **Description:** Provide a brief description of the integration.\n\n  \n\n6.  **Test and Save Integration**\n\n\t- Click `Test Connection` to ensure the provided details are correct and the connection is successful.\n\n\t- Once the connection test is successful, click `Set up source` to complete the configuration.\n\n  \n\n#### Troubleshooting\n\nIf you encounter any issues during setup:\n\n-  **Check API Token:** Ensure the provided personal api token is correct and has the necessary permissions.\n\n-  **Verify Domain URL:** Confirm the domain URL is correct.\n\n-  **Verify Username :** Confirm the username is correct.\n\n-  **Network Issues:** Check for any network issues that might be affecting connectivity.\n\n  \n\n#### Frequently Asked Questions (FAQs)\n\n1.  **How do I find my JENKINS API Token?**\n\n- To find your JENKINS API Token, Navigate to User Configuration click on Generate New Token. You can also refer to this <a href=\"https://www.baeldung.com/ops/jenkins-api-token\" target=\"_blank\" rel=\"noopener\"><span>documentation</span></a> for detailed instructions.\n\n  \n\n2.  **What should I do if my connection fails?**\n\n- Ensure the provided personal api token is correct and has the necessary permissions.\n\n- Verify the domain URL is correct.\n\n- Verify the username is correct.\n\n- Check your network connection and firewall settings.\n\n\n\n3.  **Can I customize the data being synced?**\n\n- Yes, you can customize the data being synced by adjusting the settings during the setup process.\n\n  \n\n#### Support\n\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io\" target=\"_blank\" rel=\"noopener\"><span>Support Documentation</span></a>.\n"
