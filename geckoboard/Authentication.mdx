#### API Key

Please follow up the below steps to get your API Key

1. Click on your initials in the top right hand corner.
   ![Geckoboard API Key](https://geckoboard-2.intercom-attachments-1.com/i/o/479792051/601a615923c2f58e79a2b6d4/highlight_initials.png)
2. From the dropdown, select Account.
3. On the Account Details screen, scroll down and look for API Key towards the bottom of the page.
   ![Geckoboard Account](https://geckoboard-2.intercom-attachments-1.com/i/o/479792052/03b1a299e768438656d56ec8/highlight_api_key.png)
