# Geckoboard

Use the Geckoboard connector to perform various operations such as creating a custom widget, dataset widget and more. To do this, you'll need to create a new Geckoboard API application. You can find the detailed instructions to create an Geckoboard API application in the <a href="https://developer.geckoboard.com/" target="_blank" rel="noopener"><span>Geckoboard API documentation</span></a>.

#### Prerequisites

To use the Geckoboard connector, you need to have an Geckoboard account. If you don't have one, you can create an Geckoboard account at <a href="https://www.geckoboard.com/" target="_blank" rel="noopener"><span>geckoboard.com</span></a>.
