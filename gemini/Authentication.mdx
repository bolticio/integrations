"# **Gemini Authentication**\n\nThis document outlines the setup for authenticating with the Gemini API using an API key. The authentication method ensures secure interaction with Google's Gemini, the generative AI system.\n\n## **Authentication Types**\n\n### **API Key Authentication**\n\nThe Gemini API requires an **API key** for every request. This key is used to securely authenticate and authorize API interactions with Gemini's generative models.\n\n## **Parameters**\n\n### **Authentication Type**\n\n-   **Parameter Name**: `type`\n-   **Display Name**: Authentication Type\n-   **Description**: Choose the type of authentication you want to use.\n-   **Type**: Select\n-   **Options**:\n    -   **API Key**: Use your Gemini API key for authentication.\n-   **Validation**: Required\n\n### **API Key**\n\n-   **Parameter Name**: `api_key`\n-   **Display Name**: Gemini API Key\n-   **Description**: Your Gemini API key. Refer to the <a href=\"https://ai.google.dev/api?lang=python\" target=\"_blank\">Google Gemini API documentation</a> for details.\n-   **Type**: Password\n-   **Placeholder**: Enter your API Key\n-   **Validation**: Required\n\n## **Validation**\n\nOnce the API key is entered, the connection is validated using the Gemini API.\n\n-   **Method**: POST\n-   **Headers**:\n    -   **Authorization**: Bearer `<YOUR_API_KEY>`\n    -   **Accept**: `application/json`\n    -   **Content-Type**: `application/json`\n-   **Response**:\n    -   **Success**: The response will include output from `{{response.data}}`.\n    -   **Error**: If an error occurs, the message will be displayed as `{{response.data.error.message}}`, with the corresponding status code: `{{response.status}}`.\n\n## **Troubleshooting**\n\nIf the connection fails:\n\n-   Ensure your **API Key** is correct.\n-   Confirm that your Gemini API account has the necessary permissions.\n-   Verify the network connection.\n"
