

# Mailmodo

This guide walks you through setting up the Mailmodo integration within Boltic.



## Prerequisites

To set up the Mailmodo integration, you'll need:

- A Mailmodo account

- The Mailmodo API Key



## Setting Up the Mailmodo Integration in Boltic

1. **Log In**

   - Log into your Mailmodo account.



2. **Obtain API Key**

   - Navigate to **Integrations** from the vertical menu on the left.

   - Under **Available for Connection**, locate **Make** and click the authentication CTA.

   - Copy the API Key from the displayed Make Integration window.



3. **Log into Boltic**

   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.



4. **Navigate to Integrations**

   - Click on `Sources` and then click on `+ Add New source`.



5. **Select Source Type**

   - On the Set up the source page, search for `Mailmodo` from the search bar and select it.



6. **Configure Integration**

   - **Name:** Enter a name for the Mailmodo integration.

   - **API Key:** Paste the API Key copied from Mailmodo.



7. **Details (Optional)**

   - **Tags:** Add any tags for categorization.

   - **Description:** Provide a brief description of the integration.



8. **Test and Save Integration**

   - Click `Test Connection` to ensure the provided details are correct and the connection is successful.

   - Once the connection test is successful, click `Set up source` to complete the configuration.



## Setting Up Mailmodo Webhooks

Mailmodo uses webhooks to trigger a scenario when an event occurs. To set up a webhook:

1. Add a Mailmodo instant module to your scenario and click **Create a webhook**.

2. **Webhook Name:** Enter a name for the webhook (optional).

3. **Connection:** Select the corresponding connection for the webhook.

4. Click **Save** and then **Copy address to clipboard**.

5. Log in to your Mailmodo account and follow the relevant setup instructions in the Mailmodo support articles.

6. Use the provided webhook URL in the respective workflow setup in Mailmodo.



## Frequently Asked Questions (FAQs)

1. **How do I find my Mailmodo API Key?**

   - Log into your Mailmodo account, go to Integrations, and copy the API Key from the Make Integration window. Follow the instructions provided earlier in this guide.



2. **What should I do if my connection fails?**

   - Ensure your API Key is correct.
   - Verify that the API Key is correctly entered in the Boltic configuration.
   - Check your network connection and firewall settings.



3. **How often is the data synced?**

   - The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.



4. **Can I customize the data being synced?**

   - Yes, you can customize the data being synced by adjusting the settings during the setup process.
