"\n\n# Mailmodo\n\nThis guide walks you through setting up the Mailmodo integration within Boltic.\n\n\n\n## Prerequisites\n\nTo set up the Mailmodo integration, you'll need:\n\n- A Mailmodo account\n\n- The Mailmodo API Key\n\n\n\n## Setting Up the Mailmodo Integration in Boltic\n\n1. **Log In**\n\n   - Log into your Mailmodo account.\n\n\n\n2. **Obtain API Key**\n\n   - Navigate to **Integrations** from the vertical menu on the left.\n\n   - Under **Available for Connection**, locate **Make** and click the authentication CTA.\n\n   - Copy the API Key from the displayed Make Integration window.\n\n\n\n3. **Log into Boltic**\n\n   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n\n\n4. **Navigate to Integrations**\n\n   - Click on `Sources` and then click on `+ Add New source`.\n\n\n\n5. **Select Source Type**\n\n   - On the Set up the source page, search for `Mailmodo` from the search bar and select it.\n\n\n\n6. **Configure Integration**\n\n   - **Name:** Enter a name for the Mailmodo integration.\n\n   - **API Key:** Paste the API Key copied from Mailmodo.\n\n\n\n7. **Details (Optional)**\n\n   - **Tags:** Add any tags for categorization.\n\n   - **Description:** Provide a brief description of the integration.\n\n\n\n8. **Test and Save Integration**\n\n   - Click `Test Connection` to ensure the provided details are correct and the connection is successful.\n\n   - Once the connection test is successful, click `Set up source` to complete the configuration.\n\n\n\n## Setting Up Mailmodo Webhooks\n\nMailmodo uses webhooks to trigger a scenario when an event occurs. To set up a webhook:\n\n1. Add a Mailmodo instant module to your scenario and click **Create a webhook**.\n\n2. **Webhook Name:** Enter a name for the webhook (optional).\n\n3. **Connection:** Select the corresponding connection for the webhook.\n\n4. Click **Save** and then **Copy address to clipboard**.\n\n5. Log in to your Mailmodo account and follow the relevant setup instructions in the Mailmodo support articles.\n\n6. Use the provided webhook URL in the respective workflow setup in Mailmodo.\n\n\n\n## Frequently Asked Questions (FAQs)\n\n1. **How do I find my Mailmodo API Key?**\n\n   - Log into your Mailmodo account, go to Integrations, and copy the API Key from the Make Integration window. Follow the instructions provided earlier in this guide.\n\n\n\n2. **What should I do if my connection fails?**\n\n   - Ensure your API Key is correct.\n   - Verify that the API Key is correctly entered in the Boltic configuration.\n   - Check your network connection and firewall settings.\n\n\n\n3. **How often is the data synced?**\n\n   - The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.\n\n\n\n4. **Can I customize the data being synced?**\n\n   - Yes, you can customize the data being synced by adjusting the settings during the setup process.\n\n\n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io\" target=\"_blank\" rel=\"noopener\"><span>Support Documentation</span></a>.\n"
