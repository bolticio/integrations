# Fynd Engage

This document outlines how to authenticate with Fynd Engage in Boltic using **API Key and Secret** authentication.

## Authentication Type

### Access Token

Fynd Engage uses an **Access Token** method based on your **API Key** and **API Secret**. These credentials are used to generate authorization headers that authenticate API requests.

## Required Fields

### API Key

- **Description**: Your Fynd API Key.
- **Where to Find**:  
- **Login to Fynd Engage - Settings - Generate API KEY - Name API Key - API key and Secret is Generated** 

### API Secret

- **Description**: Your Fynd API Secret.
- **Where to Find**:  
- **Login to Fynd Engage - Settings - Generate API KEY - Name API Key - API key and Secret is Generated**
