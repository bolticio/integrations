"# Fynd Engage\n\nThis document outlines how to authenticate with Fynd Engage in Boltic using **API Key and Secret** authentication.\n\n## Authentication Type\n\n### Access Token\n\nFynd Engage uses an **Access Token** method based on your **API Key** and **API Secret**. These credentials are used to generate authorization headers that authenticate API requests.\n\n## Required Fields\n\n### API Key\n\n- **Description**: Your Fynd API Key.\n- **Where to Find**:  \n- **Login to Fynd Engage - Settings - Generate API KEY - Name API Key - API key and Secret is Generated** \n\n### API Secret\n\n- **Description**: Your Fynd API Secret.\n- **Where to Find**:  \n- **Login to Fynd Engage - Settings - Generate API KEY - Name API Key - API key and Secret is Generated** \n\n---\n\n"
