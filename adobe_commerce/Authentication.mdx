# Authentication


##  Access Token Authentication

This method uses your **Admin Username and Password** to obtain a access token.

  - **Access Token:** Use your Admin Username and Password to generate a session token.  <a href="https://experienceleague.adobe.com/en/docs/commerce-admin/start/admin/admin-signin" target="_blank" rel="noopener"> <span> Documentation</span> </a>

## Troubleshooting

If the connection fails:

-   Ensure your **Admin Username and Password** is correct.
-   Confirm that your Adobe Commerce has the required access .
-   Verify your network connection.


## Conclusion

After successfully setting up your Admin Username and Password key, you should be able to authenticate and start interacting with the Adobe Commerce through Boltic.
