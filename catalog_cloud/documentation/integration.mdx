# Catalog Cloud

Use the Catalog Cloud connector to send callback to your rule. To do this, you'll need to create a rule on [Catalog Cloud](https://pim.fynd.com/). You can find the detailed instructions to create a Rule integration in the Catalog Cloud documentation.

## Prerequisites

To use the Catalog Cloud connector, you need to have an Catalog Cloud account. If you don't have one, you can create an Catalog Cloud account at [pim.fynd.com](https://pim.fynd.com/).

## Supported Authentication Methods

-   API Key

## Basic Operations

-   Response callback to rule engine
