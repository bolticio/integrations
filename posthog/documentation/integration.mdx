# PostHog

Use the PostHog connector to perform various operations such as get a project, list events, and more. To do this, you'll need to create a new PostHog application. You can find the detailed instructions to create an PostHog application in the [PostHog developers documentation](https://posthog.com/docs/api).

## Prerequisites

To use the PostHog connector, you need to have an PostHog account. If you don't have one, you can create an PostHog account at [https://us.posthog.com/signup](https://us.posthog.com/signup).

## Supported Authentication Methods

-   Personal API key

## Basic Operations

### Projects

-   Get a project
-   List all projects

### Events

-   Get an event
-   List all events

### Organizations

-   Get an organization
-   List all organizations
