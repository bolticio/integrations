

# PostHog



This guide walks you through setting up the PostHog integration within Boltic.



## Pre-requisites


To use the PostHog integration, you'll need:

- A PostHog account. If you don't have one, you can create an PostHog account at [PostHog signup page](https://us.posthog.com/signup).

- Authentication API key



## Setting Up the PostHog Integration in Boltic



1.  **Log In**

	- Log into your Boltic account or navigate to the Boltic Open Source dashboard.



2.  **Navigate to Integrations from the Primary Menu**

	- Click on `Sources` and then click on `+ Add New source`.



3.  **Select Source Type**

	- On the Set up the source page, search for `PostHog` from the search bar and select it.



4.  **Configure Integration**

	-  **Name:** Enter a name for your PostHog integration.

	-  **Authentication Type:** Select the Authentication type.

	-  **API key:** Enter your PostHog account's personal API key. To get your personal API key in PostHog, you need to follow the steps:

         - Login to your PostHog account.

         - Go to `Dashboard` and navigate to `Settings`.

         - In the settings, go into the `Personal API keys` section.

         - Click on `Create personal API key` and create your own key with all the necessary permissions and copy that.



5.  **Details (Optional)**

	-  **Tags:** Add any tags for categorization.

	-  **Description:** Provide a brief description of the integration.



6.  **Test and Save Integration**

	- Click `Test Connection` to ensure the provided details are correct and the connection is successful.

	- Once the connection test is successful, click `Set up source` to complete the configuration.


## Troubleshooting

If you encounter any issues during setup:

-  **Check API key:** Ensure the provided API key is correct and has the necessary permissions.

-  **Network Issues:** Check for any network issues that might be affecting connectivity.



## Frequently Asked Questions (FAQs)

1.  **How do I find my PostHog personal API key?**

- To find your PostHog personal API key, navigate to your PostHog account, go to `Dashboard` and navigate to `Settings`. In the settings, go into the `Personal API keys` section and create your own key with all the necessary permissions.


2.  **What should I do if my connection fails?**

- Ensure the provided API key is correct and has the necessary permissions.

- Check your network connection.



3.  **How often is the data synced?**

- The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.



4.  **Can I customize the data being synced?**

- Yes, you can customize the data being synced by adjusting the settings during the setup process.
