"\n\n# PostHog\n\n\n\nThis guide walks you through setting up the PostHog integration within Boltic.\n\n\n\n## Pre-requisites\n\n\nTo use the PostHog integration, you'll need:\n\n- A PostHog account. If you don't have one, you can create an PostHog account at [PostHog signup page](https://us.posthog.com/signup).\n\n- Authentication API key\n\n\n\n## Setting Up the PostHog Integration in Boltic\n\n\n\n1.  **Log In**\n\n\t- Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n\n\n2.  **Navigate to Integrations from the Primary Menu**\n\n\t- Click on `Sources` and then click on `+ Add New source`.\n\n\n\n3.  **Select Source Type**\n\n\t- On the Set up the source page, search for `PostHog` from the search bar and select it.\n\n\n\n4.  **Configure Integration**\n\n\t-  **Name:** Enter a name for your PostHog integration.\n\n\t-  **Authentication Type:** Select the Authentication type.\n\n\t-  **API key:** Enter your PostHog account's personal API key. To get your personal API key in PostHog, you need to follow the steps:\n\n         - Login to your PostHog account.\n\n         - Go to `Dashboard` and navigate to `Settings`.\n\n         - In the settings, go into the `Personal API keys` section.\n\n         - Click on `Create personal API key` and create your own key with all the necessary permissions and copy that.\n\n\n\n5.  **Details (Optional)**\n\n\t-  **Tags:** Add any tags for categorization.\n\n\t-  **Description:** Provide a brief description of the integration.\n\n\n\n6.  **Test and Save Integration**\n\n\t- Click `Test Connection` to ensure the provided details are correct and the connection is successful.\n\n\t- Once the connection test is successful, click `Set up source` to complete the configuration.\n\n\n## Troubleshooting\n\nIf you encounter any issues during setup:\n\n-  **Check API key:** Ensure the provided API key is correct and has the necessary permissions.\n\n-  **Network Issues:** Check for any network issues that might be affecting connectivity.\n\n\n\n## Frequently Asked Questions (FAQs)\n\n1.  **How do I find my PostHog personal API key?**\n\n- To find your PostHog personal API key, navigate to your PostHog account, go to `Dashboard` and navigate to `Settings`. In the settings, go into the `Personal API keys` section and create your own key with all the necessary permissions.\n\n\n2.  **What should I do if my connection fails?**\n\n- Ensure the provided API key is correct and has the necessary permissions.\n\n- Check your network connection.\n\n\n\n3.  **How often is the data synced?**\n\n- The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.\n\n\n\n4.  **Can I customize the data being synced?**\n\n- Yes, you can customize the data being synced by adjusting the settings during the setup process.\n\n\n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our [Support Documentation](https://docs.boltic.io).\n"
