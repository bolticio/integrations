# Jira

Use the Jira connector to perform various operations such as creating an issue, updating an issue, and more. To do this, you'll need to create a new Jira API Token. You can find the detailed instructions to create an Jira API Token in the [Jira API Token documentation](https://developer.atlassian.com/server/jira/platform/personal-access-token/).

## Prerequisites

To use the Jira connector, you need to have an Atlassian account. If you don't have one, you can create an Atlassian account at [atlassian.com](https://www.atlassian.com/).

## Supported Authentication Methods

- Personal Access Token

## Basic Operations

### Project
- List Projects

### Issue
- Create an issue
- Delete an issue
- Retrieve an issue
- Retrieve all issues
