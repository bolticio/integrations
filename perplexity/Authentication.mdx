"# Perplexity AI Integration Guide\n\nThis guide provides step-by-step instructions for integrating Perplexity AI with Boltic.\n\n---\n\n## Prerequisites\n\n- An API key of Perplexity AI.\n\n---\n\n## Setting Up Perplexity AI Integration in Boltic\n\n### 1. **Log In**\n- Access your Boltic account or the Boltic Open Source dashboard.\n\n### 2. **Open the Integrations Menu**\n- Select `Sources` from the primary menu and click `+ Add New Source`.\n\n### 3. **Choose the Source Type**\n- On the \"Select type of source you want to connect\" page, search for `Perplexity AI` in the search bar and select it.\n\n### 4. **Configure the Integration**\n- **Name:** Enter a name for the Perplexity AI integration.\n- **Authentication Type:** Choose \"API Key\" as the authentication method.\n- **API Key:** Input your Perplexity AI account's API key.\n\n### 5. **Configure the Integration (Optional Details)**\n\n- **Private Integration Toggle (Optional):** Toggle the switch to ensure the API key is private.\n- **Tags:** Add tags for easier categorization.\n- **Description:** Provide a brief description of the integration.\n\n### 6. **Test and Save**\n- Click `Test Connection` to verify the credentials and confirm the connection.\n- Once the test is successful, click `Set up source` to complete the configuration.\n\n---\n\n## Troubleshooting\n\nIf you encounter problems during setup:\n\n- **Verify the API Key:** Ensure the provided API key is correct and has the required permissions.\n- **Check Connectivity:** Look for potential network issues that could impact connectivity.\n\n---\n\n## Overview\n\nThis section outlines API usage, including authentication requirements and example requests.\n\n---\n\n## Authentication\n\nTo interact with the Perplexity AI API, authentication is required. Follow these steps:\n\n1. Visit the Console to generate an API key.\n2. Include the API key in the `Authorization` header of your API requests.\n\nFor additional details, refer to <a href='https://docs.perplexity.ai/guides/getting-started' target='_blank' rel='noopener noreferrer'>Quickstart Guide</a>.\n\n---\n\n## Support\n\nFor assistance, reach out to Boltic Support or explore our <a href='https://docs.boltic.io' target='_blank' rel='noopener noreferrer'>Support Documentation</a>."
