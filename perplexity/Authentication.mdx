# Perplexity AI Integration Guide

This guide provides step-by-step instructions for integrating Perplexity AI with Boltic.

---

## Prerequisites

- An API key of Perplexity AI.

---

## Setting Up Perplexity AI Integration in Boltic

### 1. **Log In**
- Access your Boltic account or the Boltic Open Source dashboard.

### 2. **Open the Integrations Menu**
- Select `Sources` from the primary menu and click `+ Add New Source`.

### 3. **Choose the Source Type**
- On the "Select type of source you want to connect" page, search for `Perplexity AI` in the search bar and select it.

### 4. **Configure the Integration**
- **Name:** Enter a name for the Perplexity AI integration.
- **Authentication Type:** Choose "API Key" as the authentication method.
- **API Key:** Input your Perplexity AI account's API key.

### 5. **Configure the Integration (Optional Details)**

- **Private Integration Toggle (Optional):** Toggle the switch to ensure the API key is private.
- **Tags:** Add tags for easier categorization.
- **Description:** Provide a brief description of the integration.

### 6. **Test and Save**
- Click `Test Connection` to verify the credentials and confirm the connection.
- Once the test is successful, click `Set up source` to complete the configuration.

---

## Troubleshooting

If you encounter problems during setup:

- **Verify the API Key:** Ensure the provided API key is correct and has the required permissions.
- **Check Connectivity:** Look for potential network issues that could impact connectivity.

## Authentication

To interact with the Perplexity AI API, authentication is required. Follow these steps:

1. Visit the Console to generate an API key.
2. Include the API key in the `Authorization` header of your API requests.

For additional details, refer to <a href='https://docs.perplexity.ai/guides/getting-started' target='_blank' rel='noopener noreferrer'>Quickstart Guide</a>.
