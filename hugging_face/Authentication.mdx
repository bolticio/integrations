"## Hugging Face  \n\nThis guide walks you through setting up the Hugging Face integration in Boltic using API Key authentication.  \n\n## Requirements  \n\nTo set up the Hugging Face integration, you'll need:  \n\n- A <a href=\"https://huggingface.co/join\" target=\"_blank\" rel=\"noopener\"><span>Hugging Face account</span></a>  \n- An API Key for accessing Hugging Face services (obtainable from your <a href=\"https://huggingface.co/settings/tokens\" target=\"_blank\" rel=\"noopener\"><span>Hugging Face account settings</span></a>)  \n\n## Setting Up the Hugging Face Integration in Boltic  \n\n### 1. Log In  \n\nLog into your <a href=\"https://www.boltic.io/\" target=\"_blank\" rel=\"noopener\"><span>Boltic account</span></a> or navigate to the Boltic Open Source / Destination dashboard.  \n\n### 2. Navigate to Integrations from the Primary Menu  \n\nClick on workflows and then click on `+ Add New Source / Destination`.  \n\n### 3. Select Source / Destination Type  \n\nOn the *Set up the Source / Destination* page, search for `Hugging Face` in the search bar and select it.  \n\n### 4. Configure Integration  \n\n#### **Basic Details**  \n\n- **Name:** Enter a name for the Hugging Face integration.  \n- **Description:** Add a description for your Hugging Face connection.  \n\n#### **Authentication**  \n\n1. **Authentication Type:**  \n   - Select `API Key` from the dropdown menu as the authentication type.  \n\n2. **API Key:**  \n   - Enter your Hugging Face API Key in the designated field.  \n   - Ensure you have copied the key from your <a href=\"https://huggingface.co/settings/tokens\" target=\"_blank\" rel=\"noopener\"><span>Hugging Face account settings</span></a>. If you don't have an API Key, generate one by logging into your Hugging Face account and navigating to **Settings > Access Tokens**.  \n\n### 5. Test and Save Integration  \n\n- Click `Test Connection` to ensure the provided API Key is valid and the connection is successful.  \n- Once the connection test is successful, click `Set up Source / Destination` to complete the configuration.  \n"
