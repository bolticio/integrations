## Hugging Face  

This guide walks you through setting up the Hugging Face integration in Boltic using API Key authentication.  

## Requirements  

To set up the Hugging Face integration, you'll need:  

- A <a href="https://huggingface.co/join" target="_blank" rel="noopener"><span>Hugging Face account</span></a>  
- An API Key for accessing Hugging Face services (obtainable from your <a href="https://huggingface.co/settings/tokens" target="_blank" rel="noopener"><span>Hugging Face account settings</span></a>)  

## Setting Up the Hugging Face Integration in Boltic  

### 1. Log In  

Log into your <a href="https://www.boltic.io/" target="_blank" rel="noopener"><span>Boltic account</span></a> or navigate to the Boltic Open Source / Destination dashboard.  

### 2. Navigate to Integrations from the Primary Menu  

Click on workflows and then click on `+ Add New Source / Destination`.  

### 3. Select Source / Destination Type  

On the *Set up the Source / Destination* page, search for `Hugging Face` in the search bar and select it.  

### 4. Configure Integration  

#### **Basic Details**  

- **Name:** Enter a name for the Hugging Face integration.  
- **Description:** Add a description for your Hugging Face connection.  

#### **Authentication**  

1. **Authentication Type:**  
   - Select `API Key` from the dropdown menu as the authentication type.  

2. **API Key:**  
   - Enter your Hugging Face API Key in the designated field.  
   - Ensure you have copied the key from your <a href="https://huggingface.co/settings/tokens" target="_blank" rel="noopener"><span>Hugging Face account settings</span></a>. If you don't have an API Key, generate one by logging into your Hugging Face account and navigating to **Settings > Access Tokens**.  

### 5. Test and Save Integration  

- Click `Test Connection` to ensure the provided API Key is valid and the connection is successful.  
- Once the connection test is successful, click `Set up Source / Destination` to complete the configuration.
