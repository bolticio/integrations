"## Nanonets Authentication\n\n### Requirements\nTo use the Nanonets, you need:\n\n- A Nanonets account.\n- An API Key from Nanonets (available in your account settings).\n\n### Supported Authentication Methods\n**Basic Authentication**\n\n### Authentication\nSet up Nanonets authentication using your API Key to authenticate and access the API.\n\n#### Obtaining API Key\n1. Sign up or log in to <a href=\"https://app.nanonets.com/\">Nanonets</a>.\n2. Navigate to the API dashboard.\n3. Copy your API Key from the settings page.\n\n### Handling Authentication Errors\nIf authentication fails, the API will return a **401 Unauthorized** response. Ensure that:\n\n- The API Key is correct.\n- The API Key is Base64 encoded properly.\n- The request includes the `Authorization` header.\n\n### Security Best Practices\n- Keep your API Key secure; do not expose it in public repositories.\n- Rotate API Keys periodically.\n- Use environment variables to store API Keys instead of hardcoding them.\n\n### Support\nFor more detailed instructions on setting up your Nanonets integration, refer to the Nanonets documentation or contact Nanonets support."
