## Nanonets

Use the Nanonets OCR to perform text extraction from images and documents. This AI-powered OCR automates document processing, making it useful for invoices, receipts, ID verification, and contracts.

### Requirements
To use the Nanonets OCR, you need:

- A Nanonets account.
- An API Key from Nanonets (available in your account settings).

### Supported Authentication Methods
**Basic Authentication**

### Features
- AI-based OCR with high accuracy
- Custom model training for domain-specific OCR
- Support for multiple file formats
- RESTful API for easy integration
- Multi-language support
- Exponential backoff for rate limits and retries

### Getting Started
#### Create an Account & Get API Key
1. Sign up at <a href="https://app.nanonets.com/" target="_blank">Nanonets</a>.
2. Navigate to the API dashboard.
3. Get your API Key from the settings page.

### Supported File Formats
- **Images**: JPEG / JPG, PNG, TIFF / TIF, BMP, GIF
- **Documents**: PDF (multi-page supported)

### Use Cases
- Invoice & Receipt Processing  
- Automated ID Verification  
- Document Digitization  
- Bank Statements & Financial Reports
