# betterstack

Use the betterstack connector to perform various operations such as check all monitors,check status pages and more. To do this, you'll need to create a new betterstack account. You can find the detailed instructions to create a betterstack account in the [betterstack  documentation](https://betterstack.com/docs/).

## Prerequisites

To use the betterstack connector, you need to have a betterstack account. If you don't have one, you can create a betterstack account at [login.betterstack.com](https://betterstack.com/users/sign-up).

## Supported Authentication Methods

-   API key

## Basic Operations

### uptime

-   List all existing monitors
-   List monitor by ID
-   Get a monitor's response time
-   Get a monitor's availability summary

### status

-   Get a single status page
-   Get all existing status page
