"# Clickhouse Configuration\n\nThis guide walks you through setting up the Clickhouse integration in Boltic.\n\n## Requirements\n\nTo set up the Clickhouse integration, you'll need:\n\n-   A Clickhouse server connection\n-   Clickhouse server credentials (username and password or access key)\n-   **Optional:** SSH configuration (for connecting securely via SSH)\n\n## Setting Up the Clickhouse Integration in Boltic\n\n### 1. Log In\n\nLog into your Boltic account or navigate to the Boltic Open Source / Destination dashboard.\n\n### 2. Navigate to Integrations from the Primary Menu\n\nClick on `Source / Destinations` and then click on `+ Add New Source / Destination`.\n\n### 3. Select Source / Destination Type\n\nOn the Set up the Source / Destination page, search for `Clickhouse` from the search bar and select it.\n\n### 4. Configure Integration\n\n#### **Basic Details**\n\n-   **Name:** Enter a name for the Clickhouse integration.\n-   **Description:** Add a description for your Clickhouse connection.\n\n#### **Connection Details**\n\n-   **Host:** Enter the address of the Clickhouse server. This should be a publicly accessible IP address or domain. Avoid using `localhost` or `127.0.0.1` unless connecting locally.\n-   **Port:** Enter the Clickhouse server's port number (default is 9000).\n-   **Database Name:** Enter the name of the database you want to connect to.\n-   **Username:** Enter the Clickhouse username if authentication is required.\n-   **Password:** Enter the password associated with the Clickhouse user if authentication is required.\n\n#### **SSH Configuration (Optional)**\n\nToggle `SSH Configuration` on if you need to connect via SSH.\n\n-   **SSH Host:** Enter the SSH host address.\n-   **SSH Port:** Enter the port for SSH.\n-   **SSH Username:** Enter the SSH username.\n-   **SSH Authentication Method:** Select either `Private Key` or `Password` for authentication.\n\n    -   If you choose **Private Key**, upload the private key file.\n    -   If you choose **Password**, enter the SSH password.\n\n### 5. Test and Save Integration\n\n-   Click `Test Connection` to ensure the provided details are correct and the connection is successful.\n-   Once the connection test is successful, click `Set up Source / Destination` to complete the configuration.\n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io/\" target=\"_blank\" rel=\"noopener\"><span>Support Documentation</span></a>.\n"
