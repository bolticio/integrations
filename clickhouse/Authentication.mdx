# Clickhouse Configuration

This guide walks you through setting up the Clickhouse integration in Boltic.

## Requirements

To set up the Clickhouse integration, you'll need:

-   A Clickhouse server connection
-   Clickhouse server credentials (username and password or access key)
-   **Optional:** SSH configuration (for connecting securely via SSH)

## Setting Up the Clickhouse Integration in Boltic

### 1. Log In

Log into your Boltic account or navigate to the Boltic Open Source / Destination dashboard.

### 2. Navigate to Integrations from the Primary Menu

Click on `Source / Destinations` and then click on `+ Add New Source / Destination`.

### 3. Select Source / Destination Type

On the Set up the Source / Destination page, search for `Clickhouse` from the search bar and select it.

### 4. Configure Integration

#### **Basic Details**

-   **Name:** Enter a name for the Clickhouse integration.
-   **Description:** Add a description for your Clickhouse connection.

#### **Connection Details**

-   **Host:** Enter the address of the Clickhouse server. This should be a publicly accessible IP address or domain. Avoid using `localhost` or `127.0.0.1` unless connecting locally.
-   **Port:** Enter the Clickhouse server's port number (default is 9000).
-   **Database Name:** Enter the name of the database you want to connect to.
-   **Username:** Enter the Clickhouse username if authentication is required.
-   **Password:** Enter the password associated with the Clickhouse user if authentication is required.

#### **SSH Configuration (Optional)**

Toggle `SSH Configuration` on if you need to connect via SSH.

-   **SSH Host:** Enter the SSH host address.
-   **SSH Port:** Enter the port for SSH.
-   **SSH Username:** Enter the SSH username.
-   **SSH Authentication Method:** Select either `Private Key` or `Password` for authentication.

    -   If you choose **Private Key**, upload the private key file.
    -   If you choose **Password**, enter the SSH password.

### 5. Test and Save Integration

-   Click `Test Connection` to ensure the provided details are correct and the connection is successful.
-   Once the connection test is successful, click `Set up Source / Destination` to complete the configuration.
