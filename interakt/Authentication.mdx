# Interakt Authentication

This document outlines the setup for Interakt authentication in Boltic using Basic Authentication. The authentication method can be selected and configured through the following steps.

This guide walks you through setting up the Interakt integration within Boltic.

## Prerequisites

To set up the Interakt integration, you'll need:

-   The Interakt api token.

## Setting Up the Interakt Integration in Boltic

1.  **Log In**

    -   Log into your Boltic account or navigate to the Boltic Open Source dashboard.

2.  **Navigate to Integrations from the Primary Menu**

    -   Click on `Sources` and then click on `+ Add New source`.

3.  **Select Source Type**

    -   On the Set up the source page search for `Interakt` from the search bar and select it.

4.  **Configure Integration**

    -   **Name:** Enter a name for the Interakt integration.

    -   **Auth type:** Select Authentication type to Api Key.

    -   **Api Key:** Enter your Interakt personal api key. The Interakt integration currently supports only the "Api Key" authentication type.

5.  **Details (Optional)**

    -   **Tags:** Add any tags for categorization.

    -   **Description:** Provide a brief description of the integration.

6.  **Test and Save Integration**

    -   Click `Test & Save` to ensure the provided details are correct and the connection is successful.

## Troubleshooting

If you encounter any issues during setup:

-   **Check Api Key:** Ensure the provided api key is correct and has the necessary permissions.

-   **Network Issues:** Check for any network issues that might be affecting connectivity.

## Frequently Asked Questions (FAQs)

1.  **How do I find my Interakt api key?**

-   To find your Interakt api key, navigate to your Interakt account settings and look for the Organization Name section (at top of the left side panel).

-   After clicking on the organization name you would see the option named as Developer Settings in that you can find the Api key named as Secret Key.

2.  **What should I do if my connection fails?**

-   Ensure your api token is correct.

-   Check your network connection and firewall settings.
