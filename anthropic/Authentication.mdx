"# Anthropic\n\nThis document outlines the setup for authenticating with the Anthropic API using an API key. The authentication method ensures secure interaction with Claude, Anthropic's AI system.\n\n## Authentication Types\n\n### API Key Authentication\n\nThe Anthropic API requires an **x-api-key** header for every request. This key is used to securely authenticate and authorize API interactions.\n\n### Steps to create API key\n\n1. Sign up/Login to Anthropic <a href=\"https://console.anthropic.com/dashboard\" target=\"_blank\">Console </a>\n2. Go to API Keys under Account Settings.\n3. Click \"Create API Key\" and copy it.\n4. Store securely (Do not share publicly).\n\n### Parameters\n\n#### Authentication Type\n\n-   **Parameter Name:** `type`\n-   **Display Name:** Authentication Type\n-   **Description:** Choose the type of authentication you want to use.\n-   **Type:** Select\n-   **Options:**\n    -   **API Key:** Use your Anthropic API key for authentication.\n-   **Validation:** Required\n\n#### API Key\n\n-   **Parameter Name:** `api_key`\n-   **Display Name:** Anthropic API Key\n-   **Description:** Your Anthropic API key. Refer to the <a href=\"https://docs.anthropic.com/en/api/getting-started\" target=\"_blank\">Anthropic API documentation</a> for details.\n-   **Type:** Password\n-   **Placeholder:** Enter your API Key\n-   **Validation:** Required\n\n### Validation\n\nOnce the API key is entered, the connection is validated.\n\n## Troubleshooting\n\nIf the connection fails:\n\n-   Ensure your **API Key** is correct.\n-   Confirm that your Anthropic account has the necessary permissions.\n-   Verify the network connection.\n"
