# Anthropic

This document outlines the setup for authenticating with the Anthropic API using an API key. The authentication method ensures secure interaction with Claude, Anthropic's AI system.

## Authentication Types

### API Key Authentication

The Anthropic API requires an **x-api-key** header for every request. This key is used to securely authenticate and authorize API interactions.

### Steps to create API key

1. Sign up/Login to Anthropic <a href="https://console.anthropic.com/dashboard" target="_blank">Console </a>
2. Go to API Keys under Account Settings.
3. Click "Create API Key" and copy it.
4. Store securely (Do not share publicly).

### Parameters

#### Authentication Type

-   **Parameter Name:** `type`
-   **Display Name:** Authentication Type
-   **Description:** Choose the type of authentication you want to use.
-   **Type:** Select
-   **Options:**
    -   **API Key:** Use your Anthropic API key for authentication.
-   **Validation:** Required

#### API Key

-   **Parameter Name:** `api_key`
-   **Display Name:** Anthropic API Key
-   **Description:** Your Anthropic API key. Refer to the <a href="https://docs.anthropic.com/en/api/getting-started" target="_blank">Anthropic API documentation</a> for details.
-   **Type:** Password
-   **Placeholder:** Enter your API Key
-   **Validation:** Required

### Validation

Once the API key is entered, the connection is validated.

## Troubleshooting

If the connection fails:

-   Ensure your **API Key** is correct.
-   Confirm that your Anthropic account has the necessary permissions.
-   Verify the network connection.
