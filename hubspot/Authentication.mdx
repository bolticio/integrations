# HubSpot Authentication

## Overview

HubSpot's free CRM powers your customer support, sales, and marketing with easy-to-use features like live chat, meeting scheduling, and email tracking.

## Authentication Ways

-   Private App Access Token

## Private App Access Token

To authenticate with HubSpot using a Private App, you need to create a Private App and generate an Access Token.

**Steps:**
1. Log in to your HubSpot account.
2. Go to **Settings** > **Integrations** > **Private Apps**.
3. Click **Create a private app**.
4. Enter a name and description for your app.
5. **Add the following scopes to your Private App:**

    ```
    crm.objects.subscriptions.write
    crm.objects.line_items.read
    crm.objects.line_items.write
    crm.schemas.line_items.read
    forms
    tickets
    communication_preferences.read
    crm.lists.read
    crm.objects.contacts.read
    crm.objects.subscriptions.read
    crm.schemas.subscriptions.read
    crm.schemas.commercepayments.read
    crm.objects.commercepayments.read 
    crm.objects.contacts.write
    crm.objects.users.write
    crm.objects.marketing_events.read
    crm.objects.marketing_events.write
    crm.objects.companies.write
    crm.schemas.contacts.read
    crm.pipelines.orders.write
    crm.pipelines.orders.read
    crm.objects.leads.read
    crm.objects.leads.write
    crm.lists.write
    crm.objects.companies.read
    crm.objects.deals.read
    crm.objects.deals.write
    crm.schemas.companies.read
    crm.schemas.companies.write
    crm.schemas.contacts.write
    crm.schemas.deals.read
    crm.schemas.deals.write
    crm.objects.owners.read
    settings.users.read
    crm.objects.users.read
    ```

6. Click **Create app** and copy your Access Token. Store it securely.

For more details, see the <a href="https://developers.hubspot.com/beta-docs/guides/apps/private-apps/overview" target="_blank" rel="noopener"><span>HubSpot Private App documentation</span></a>
