"# TMS Authentication\n\nGuide on authenticating API requests for the Transportation Management System (TMS).\n\nThis document provides a comprehensive guide on how to authenticate API requests for the Transportation Management System (TMS). The system supports API Key-based authentication to ensure secure access to its services.\n\n## Authentication Types\n\nTMS offers different methods of authentication to cater to various integration scenarios. Currently, the system supports the following authentication type:\n\n- **API Key**: A secure and simple way to authenticate API requests.\n\n## API Key Authentication\n\nTo use API Key authentication, you must include your API key in the request headers. This method is preferred for its simplicity and effectiveness in protecting access.\n\n### Obtaining Your API Key\n\n1. Log in to your <a href=\"https://scm.fynd.com/tms/\" target=\"_blank\">Fynd TMS</a> account dashboard.\n2. Navigate to the **API** section.\n3. Generate or copy your existing API key.\n\n## **Troubleshooting**\n\nIf the connection fails:\n- Ensure your **API Key** is correct.\n- Confirm that your TMS API account has the necessary permissions.\n- Verify the network connection."
