# TMS Authentication

Guide on authenticating API requests for the Transportation Management System (TMS).

This document provides a comprehensive guide on how to authenticate API requests for the Transportation Management System (TMS). The system supports API Key-based authentication to ensure secure access to its services.

## Authentication Types

TMS offers different methods of authentication to cater to various integration scenarios. Currently, the system supports the following authentication type:

- **API Key**: A secure and simple way to authenticate API requests.

## API Key Authentication

To use API Key authentication, you must include your API key in the request headers. This method is preferred for its simplicity and effectiveness in protecting access.

### Obtaining Your API Key

1. Log in to your <a href="https://scm.fynd.com/tms/" target="_blank">Fynd TMS</a> account dashboard.
2. Navigate to the **API** section.
3. Generate or copy your existing API key.

## **Troubleshooting**

If the connection fails:
- Ensure your **API Key** is correct.
- Confirm that your TMS API account has the necessary permissions.
- Verify the network connection.
