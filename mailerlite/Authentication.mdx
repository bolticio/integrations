"# MailerLite\n\nThis guide will walk you through the steps to integrate MailerLite into Boltic.\n\n\n\n## Prerequisites\n\nBefore integrating MailerLite with Boltic, ensure you have:\n\n- A MailerLite account\n\n- The MailerLite API Key\n\n\n\n## Setting Up the MailerLite Integration in Boltic\n\n1. **Log In**\n\n   - Log into your MailerLite account.\n\n\n\n2. **Obtain API Key**\n\n   - Navigate to **Account Settings**.\n\n   - Select **Integrations** from the left-hand menu.\n\n   - Under the **Developer API** section, find your API Key or click **Generate new API Key**.\n\n   - Copy the API Key to use in Boltic.\n\n\n\n3. **Log into Boltic**\n\n   - Log into your Boltic account and access the Boltic dashboard.\n\n\n\n4. **Navigate to Integrations**\n\n   - Click on `Sources` in the dashboard and then click on `+ Add New Source`.\n\n\n\n5. **Select Source Type**\n\n   - On the “Set up the source” page, search for `MailerLite` from the search bar and select it.\n\n\n\n6. **Configure Integration**\n\n   - **Name:** Provide a name for the MailerLite integration.\n\n   - **API Key:** Paste the API Key copied from MailerLite.\n\n\n\n7. **Details (Optional)**\n\n   - **Tags:** Add tags for organizing the integration.\n\n   - **Description:** Include a brief description of the MailerLite integration.\n\n\n\n8. **Test and Save Integration**\n\n   - Click `Test Connection` to verify the API key and ensure the connection is successful.\n\n   - Once the connection is confirmed, click `Set up source` to finalize the configuration.\n\n\n\n## Frequently Asked Questions (FAQs)\n\n1. **Where do I find my MailerLite API Key?**\n\n   - You can find your API Key under **Account Settings** > **Integrations** > **Developer API** in your MailerLite account.\n\n\n\n2. **What should I do if my connection fails?**\n\n   - Ensure the API Key is correctly copied from MailerLite.\n\n   - Check your network or firewall settings to allow external connections.\n\n\n\n3. **How frequently is data synced?**\n\n   - Data sync frequency is determined by the sync settings you configure during setup.\n\n   \n\n4. **Can I customize the data being synced?**\n\n   - Yes, you can customize the data being synced based on your integration requirements.\n\n   \n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io\" target=\"_blank\" rel=\"noopener\"><span>Support Documentation</span></a>.\n"
