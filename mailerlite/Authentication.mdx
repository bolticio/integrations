# MailerLite

This guide will walk you through the steps to integrate MailerLite into Boltic.



## Prerequisites

Before integrating MailerLite with Boltic, ensure you have:

- A MailerLite account

- The MailerLite API Key



## Setting Up the MailerLite Integration in Boltic

1. **Log In**

   - Log into your MailerLite account.



2. **Obtain API Key**

   - Navigate to **Account Settings**.

   - Select **Integrations** from the left-hand menu.

   - Under the **Developer API** section, find your API Key or click **Generate new API Key**.

   - Copy the API Key to use in Boltic.



3. **Log into Boltic**

   - Log into your Boltic account and access the Boltic dashboard.



4. **Navigate to Integrations**

   - Click on `Sources` in the dashboard and then click on `+ Add New Source`.



5. **Select Source Type**

   - On the “Set up the source” page, search for `MailerLite` from the search bar and select it.



6. **Configure Integration**

   - **Name:** Provide a name for the MailerLite integration.

   - **API Key:** Paste the API Key copied from MailerLite.



7. **Details (Optional)**

   - **Tags:** Add tags for organizing the integration.

   - **Description:** Include a brief description of the MailerLite integration.



8. **Test and Save Integration**

   - Click `Test Connection` to verify the API key and ensure the connection is successful.

   - Once the connection is confirmed, click `Set up source` to finalize the configuration.



## Frequently Asked Questions (FAQs)

1. **Where do I find my MailerLite API Key?**

   - You can find your API Key under **Account Settings** > **Integrations** > **Developer API** in your MailerLite account.



2. **What should I do if my connection fails?**

   - Ensure the API Key is correctly copied from MailerLite.

   - Check your network or firewall settings to allow external connections.



3. **How frequently is data synced?**

   - Data sync frequency is determined by the sync settings you configure during setup.

   

4. **Can I customize the data being synced?**

   - Yes, you can customize the data being synced based on your integration requirements.
