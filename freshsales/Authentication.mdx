

# Freshsales



This guide walks you through setting up the Freshsales integration within Boltic.



## Prerequisites

To set up the Freshsales integration, you'll need:

- A Freshsales account

- The Freshsales domain name

- The Freshsales API Key



## Setting Up the Freshsales Integration in Boltic



1. **Log In**

   - Log into your Freshsales account dashboard.



2. **Obtain API Key**

   - Click on your **Profile Icon** > **Settings** > **API Settings**.

   - In the API Settings tab, complete the authentication to view your API Key.

   - Copy the API Key to your clipboard.



3. **Log into Boltic**

   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.



4. **Navigate to Integrations**

   - Click on `Sources` and then click on `+ Add New source`.



5. **Select Source Type**

   - On the Set up the source page, search for `Freshsales` from the search bar and select it.



6. **Configure Integration**

   - **Name:** Enter a name for the Freshsales integration.

   - **Domain:** Enter your Freshsales domain name. For example, if your Freshsales URL is `www.xyz.myfreshworks.com/crm/sales`, your domain is `xyz`.

   - **API Key:** Paste the API Key copied from the Freshsales API Settings.



7. **Details (Optional)**

   - **Tags:** Add any tags for categorization.

   - **Description:** Provide a brief description of the integration.



8. **Test and Save Integration**

   - Click `Test Connection` to ensure the provided details are correct and the connection is successful.

   - Once the connection test is successful, click `Set up source` to complete the configuration.



## Setting Up Freshsales Webhooks



Freshsales uses webhooks to trigger a scenario when an event occurs. To set up a webhook:


1. Add a Freshsales instant module to your scenario and click **Create a webhook**.


2. **Webhook Name:** Enter a name for the webhook (optional).


3. **Connection:** Select the corresponding connection for the webhook.


4. Click **Save** and then **Copy address to clipboard**.


5. Log in to your Freshsales account using <a href="https://support.freshsales.io/support/solutions/articles/50000003583-how-to-configure-a-workflow-"  target="_blank" rel="noopener"><span>Login to the Freshsales Account</span></a>.

1. **How do I find my Freshsales API Key?**

   - Navigate to your Freshsales account settings and look for the API Settings tab to find your API Key. Follow the instructions provided earlier in this guide.



2. **What should I do if my connection fails?**

   - Ensure your API Key is correct.

   - Verify the domain name and ensure there are no typos.

   - Check your network connection and firewall settings.



3. **How often is the data synced?**

   - The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.



4. **Can I customize the data being synced?**

   - Yes, you can customize the data being synced by adjusting the settings during the setup process.
