# LINE Messenger

## Overview

The LINE Messaging API allows developers to build rich conversational experiences on the LINE platform. Authentication is handled using **Channel Access Tokens**, which authorize your bot or application to interact with LINE users securely.

## Authentication Ways

- Channel Access Token

## Channel Access Token

To authenticate with the LINE Messaging API, you need a Channel Access Token from your LINE Messaging API channel. This token authorizes API requests and links them to your specific bot.

**Steps:**

1. **Log in to LINE Business Console**  
   - Go to <a href="https://developers.line.biz/console/" target="_blank" rel="noopener">LINE Business Console</a> and log in using your LINE account.

2. **Select Your Provider and Channel**  
   - Choose your **Provider** and select the **Messaging API Channel** you want to use.

3. **Get the Channel Access Token**  
   - Navigate to the **Messaging API** tab  
   - Under **Channel access token**, click **Issue** to generate or **Copy** to retrieve an existing token

For more details, visit the <a href="https://developers.line.biz/en/docs/messaging-api/overview/" target="_blank" rel="noopener">LINE Messaging API documentation</a>.
