"# Keka Authentication\n\nThis guide walks you through setting up the Keka integration within Boltic.\n\n## Prerequisites\n\nTo set up the Keka integration, you'll need:\n\n-   An keka account (Client ID, Client Secret, API Key)\n-   Following is he link to know more about client id, client secret and api key and how to use them to create new access token <a href='https://developers.keka.com/reference/authentication' target='_blank' rel=\"noopener\">Authentication Documentation</a>.\n\nIf you don't have an Keka account, you can create one at <a href=\"https://www.keka.com/signup\" target=\"_blank\" rel=\"noopener\"><span>Keka Create Account</span></a>.\n\n## Setting Up the Keka Integration in Boltic\n\n1. **Log In**\n\n    - Log into your Keka account.\n\n2. **Log into Boltic**\n\n    - Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n3. **Navigate to Integrations**\n\n    - Click on `Sources` and then click on `+ Add New source`.\n\n4. **Select Source Type**\n\n    - On the Set up the source page, search for `Keka` from the search bar and select it.\n\n5. **Basic Details**\n\n    - **Name:** Enter a name for the Keka integration.\n\n    - **Description:** Provide a brief description of the integration.\n\n6. **Connection Details**\n\n    - **Authentication Type:** Select authentication type.\n\n    - **Company:** Provide the company name.\n\n    - **Environment:** Select the environment you want to use.\n\n    - **Client ID:** Provide the client id.\n\n    - **Client Secret:** Provide the client secret.\n\n    - **API Key:** Provide the api key.\n\n7. **Test and Save Integration**\n\n    - Click `Test Connection` to ensure the connection is successful.\n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io\" target=\"_blank\" rel=\"noopener\"><span>Boltic Support Documentation</span></a>.\n"
