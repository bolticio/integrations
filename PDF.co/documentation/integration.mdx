# PDF.co
The PDF.co node allows you to integrate with the PDF.co API to automate document processing workflows, manage PDF operations, and handle file conversions. This guide helps you configure and start using the node efficiently.

## Getting Started

Before using the PDF.co node, complete the following setup.

### Prerequisites
- **PDF.co Account**: <a href="https://PDF.co/" target="_blank" rel="noopener">Sign up here</a>
- **PDF.co API Key**: <a href="https://app.PDF.co/" target="_blank" rel="noopener">Get it from your PDF.co dashboard</a>

### Supported Authentication Methods

The PDF.co node supports the following authentication methods:

- **API Key Authentication**: Authenticate using your PDF.co API key for simple and straightforward authentication.

## Basic Operations

Here are some of the basic operations you can perform with the PDF.co node:

### PDF Conversion

- **PDF to CSV**: Convert PDF tables and data into CSV format for spreadsheet analysis.
- **PDF to JSON**: Extract structured data from PDFs in JSON format for web applications.
- **PDF to Text**: Extract all text content from PDF documents for text processing.
- **PDF to Excel**: Convert PDF tables into editable Excel spreadsheets for data manipulation.
- **PDF to HTML**: Transform PDF documents into HTML format for web display.
- **PDF to XML**: Convert PDF content into XML format for data exchange.
- **PDF to JPG**: Convert PDF pages into high-quality JPG image format.
- **PDF to PNG**: Convert PDF pages into high-quality PNG image format.
- **PDF to WEBP**: Convert PDF pages into WEBP image format.
- **PDF to TIFF**: Convert PDF pages into TIFF image format.
- **DOC to PDF**: Create PDFs from DOC files.
- **CSV to PDF**: Generate PDFs from CSV files.
- **Image to PDF**: Generate PDFs from JPG, PNG, and TIFF image files.
- **URL to PDF**: Convert web pages and URLs into PDF documents.
- **HTML to PDF**: Transform HTML content into PDF format.
- **Email to PDF**: Convert email files (MSG or EML) into PDF documents.

### PDF Merging

- **Merge PDFs**: Combine multiple PDF files into a single document.
- **Merge Various Document Type**: Merge various document formats (PDF, DOC, XLS, images) into one PDF.

### PDF Splitting

- **Split PDF**: Split a PDF into multiple PDF files using page indexes or page ranges.
- **Split PDF by Text Search**: Split a PDF into multiple PDF files using page indexes or page ranges.


### Excel Conversion

- **Excel to PDF**: Convert XLS and XLSX files into PDF format.
- **Excel to CSV**: Transform Excel spreadsheets into CSV format.
- **Excel to JSON**: Convert Excel data into structured JSON format.
- **Excel to HTML**: Transform Excel files into HTML tables for web display.
- **Excel to XML**: Convert Excel data into XML format for data exchange.
- **Excel to Text**: Convert Excel data into text.

## Conclusion
The PDF.co node streamlines your document processing workflows by connecting your applications directly with PDF.co. With just a few steps, you can automate complex document operations including conversion, merging, splitting, and data extraction at scale.
