# Typeform

Use the Typeform activity to interact with your Typeform account. This activity allows you to manage forms, responses, webhooks, and images, providing seamless integration and automation with Typeform. To use this activity, you'll need an account with Typeform.

## Authentication Type

- Access Token

Refer to [Typeform's API documentation](https://developer.typeform.com/) for more information on obtaining and using your access token.

## Basic Operations

### Forms

- Create a Form: Set up a new form with custom fields, themes, and settings.
- Get a Form: Retrieve the details of a specific form.
- Update a Form: Modify an existing formâ€™s structure or settings.
- Delete a Form: Remove a form from your account.
- List Forms: Retrieve a list of all your forms.

### Responses

- List Responses: Retrieve responses from a specific form.

### Webhooks

- Create/Update a Webhook: Set up or modify a webhook to get real-time form submission notifications.
- Delete a Webhook: Remove a webhook from a form.
- List Webhooks: Retrieve a list of all webhooks associated with a form.

### Images

- Create an Image: Upload a new image to your Typeform account.
- Delete an Image: Remove an image from your account.
- List Images: Retrieve a list of all images uploaded to your account.
