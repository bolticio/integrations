# Typeform



This guide walks you through setting up the Typeform integration within Boltic.



## Prerequisites

To set up the Typeform integration, you'll need:

- A Typeform account

If you don't have a Typeform account, you can create one at <a href="https://www.typeform.com/signup/" target="_blank" rel="noopener"><span>Typeform Create Account</span></a>.



## Setting Up the Typeform Integration in Boltic

1. **Log In**

   - Log into your Typeform account.



2. **Log into Boltic**

   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.



3. **Navigate to Integrations**

   - Click on `Sources` and then click on `+ Add New source`.



4. **Select Source Type**

   - On the Set up the source page, search for `Typeform` from the search bar and select it.



5. **Authenticate Typeform Account**

   - Click `Connect to Typeform`.

   - You will be redirected to the Typeform website.

   - If you are already logged into Typeform, the connection will be automatically established. If not, log into your Typeform account.



6. **Configure Integration**

   - **Name:** Enter a name for the Typeform integration.

   - **Workspace:** Select the workspace you want to connect.



7. **Details (Optional)**

   - **Tags:** Add any tags for categorization.

   - **Description:** Provide a brief description of the integration.



8. **Test and Save Integration**

   - Click `Test Connection` to ensure the connection is successful.

   - Once the connection test is successful, click `Set up source` to complete the configuration.
