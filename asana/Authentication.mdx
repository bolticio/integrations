

# Asana



This guide walks you through setting up the Asana integration within Boltic.



## Prerequisites

To set up the Asana integration, you'll need:

- An Asana account

If you don't have an Asana account, you can create one at <a href="https://asana.com/create-account" target="_blank" rel="noopener"><span>Asana Create Account</span></a>.



## Setting Up the Asana Integration in Boltic

1. **Log In**

   - Log into your Asana account.



2. **Log into Boltic**

   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.



3. **Navigate to Integrations**

   - Click on `Sources` and then click on `+ Add New source`.



4. **Select Source Type**

   - On the Set up the source page, search for `Asana` from the search bar and select it.



5. **Authenticate Asana Account**

   - Click `Connect to Asana`.

   - You will be redirected to the Asana website.

   - If you are already logged into Asana, the connection will be automatically established. If not, log into your Asana account.



6. **Configure Integration**

   - **Name:** Enter a name for the Asana integration.

   - **Workspace:** Select the workspace you want to connect.



7. **Details (Optional)**

   - **Tags:** Add any tags for categorization.

   - **Description:** Provide a brief description of the integration.

   

8. **Test and Save Integration**

   - Click `Test Connection` to ensure the connection is successful.

   - Once the connection test is successful, click `Set up source` to complete the configuration.
