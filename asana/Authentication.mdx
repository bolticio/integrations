"\n\n# Asana\n\n\n\nThis guide walks you through setting up the Asana integration within Boltic.\n\n\n\n## Prerequisites\n\nTo set up the Asana integration, you'll need:\n\n- An Asana account\n\nIf you don't have an Asana account, you can create one at <a href=\"https://asana.com/create-account\" target=\"_blank\" rel=\"noopener\"><span>Asana Create Account</span></a>.\n\n\n\n## Setting Up the Asana Integration in Boltic\n\n1. **Log In**\n\n   - Log into your Asana account.\n\n\n\n2. **Log into Boltic**\n\n   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n\n\n3. **Navigate to Integrations**\n\n   - Click on `Sources` and then click on `+ Add New source`.\n\n\n\n4. **Select Source Type**\n\n   - On the Set up the source page, search for `Asana` from the search bar and select it.\n\n\n\n5. **Authenticate Asana Account**\n\n   - Click `Connect to Asana`.\n\n   - You will be redirected to the Asana website.\n\n   - If you are already logged into Asana, the connection will be automatically established. If not, log into your Asana account.\n\n\n\n6. **Configure Integration**\n\n   - **Name:** Enter a name for the Asana integration.\n\n   - **Workspace:** Select the workspace you want to connect.\n\n\n\n7. **Details (Optional)**\n\n   - **Tags:** Add any tags for categorization.\n\n   - **Description:** Provide a brief description of the integration.\n\n   \n\n8. **Test and Save Integration**\n\n   - Click `Test Connection` to ensure the connection is successful.\n\n   - Once the connection test is successful, click `Set up source` to complete the configuration.\n\n\n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io\" target=\"_blank\" rel=\"noopener\"><span>Support Documentation</span></a>.\n\n"
