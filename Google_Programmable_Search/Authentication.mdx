"# Google Programmable Search Engine Authentication\n\n## Overview\n\nGoogle Programmable Search Engine (formerly Custom Search Engine) allows you to build a tailored search experience powered by Google's search technology. You can access search results programmatically using the Custom Search JSON API.\n\n## Authentication Ways\n\n- API Key + Search Engine ID\n\n## API Key + Search Engine ID\n\nTo authenticate with Google Programmable Search Engine, you need to create a search engine and enable the Custom Search API in Google Cloud to obtain both an API Key and a Search Engine ID.\n\n**Steps:**\n\n1. **Create a Programmable Search Engine**\n    - Go to <a href=\"https://programmablesearchengine.google.com/\" target=\"_blank\" rel=\"noopener\"><span>Programmable Search Engine</span></a>\n    - Click **“Create a search engine”**\n    - Under **Sites to search**, you can use `www.example.com`, or `*.com` to search the entire web\n    - Click **Create**\n    - After creation, go to **Control Panel**:\n        - Enable **“Search the entire web”** under “Sites to Search”\n\n2. **Get Your Search Engine ID (cx)**\n    - In the Control Panel, go to **Basics**\n    - Copy the **Search engine ID** — this is your `cx`\n\n3. **Enable the Custom Search API in Google Cloud**\n    - Go to <a href=\"https://console.cloud.google.com/\" target=\"_blank\" rel=\"noopener\"><span>Google Cloud Console</span></a>\n    - Create a new project (or select an existing one)\n    - Navigate to **APIs & Services > Library**\n    - Search for **Custom Search API** and click **Enable**\n\n4. **Generate Your API Key**\n    - In the same project, go to **APIs & Services > Credentials**\n    - Click **Create Credentials > API key**\n\nFor more details, see the <a href=\"https://developers.google.com/custom-search/v1/overview\" target=\"_blank\" rel=\"noopener\"><span>Custom Search JSON API documentation</span></a>"
