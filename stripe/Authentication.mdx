"This guide walks you through setting up the Stripe integration within Boltic.\n\n#### Prerequisites\n\nTo set up the Stripe integration, you'll need:\n\n- The Stripe account email address\n- The Stripe API key (Secret Key)\n- Admin access to the Stripe account\n\n#### Setting Up the Stripe Integration in Boltic\n\n#### Log In\n\nLog into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n#### Navigate to Integrations from the Primary Menu\n\nClick on `Sources` and then click on `+ Add New source`.\n\n#### Select Source Type\n\nOn the Set up the source page, search for `Stripe` from the search bar and select it.\n\n#### Configure Integration\n\n- **Name:** Enter a name for the Stripe integration.\n- **Email:** Enter the email address associated with your Stripe account.\n- **API Key (Secret Key):** Enter your Stripe API key. The Stripe integration uses the Secret Key for authentication (You can get the access key from <a href=\"https://dashboard.stripe.com/\" target=\"_blank\" rel=\"noopener\">Stripe dashboard</a>).\n\n#### Obtaining Your Stripe API Key (Secret Key)\n\nTo integrate with Stripe, you'll need your Secret Key for authentication. Follow these steps to obtain it:\n\n1. **Log in to Stripe:**\n   - Go to the <a href=\"https://dashboard.stripe.com/\" target=\"_blank\" rel=\"noopener\">Stripe dashboard</a> and log in with your credentials.\n\n2. **Navigate to API Keys:**\n   - Once logged in, navigate to the **Developers** section in the left-hand menu.\n\n3. **View API Keys:**\n   - In the Developers section, click on **API keys**. Here you will see your **Publishable Key** and **Secret Key**.\n\n4. **Reveal Secret Key:**\n   - By default, the Secret Key may be partially obscured. Click on the \"Reveal test key\" or \"Reveal live key\" button next to the Secret Key to see the full key.\n\n5. **Copy the Secret Key:**\n   - Copy the Secret Key and use it in your application for authentication.\n\n#### Important Notes\n\n- **Test Mode and Live Mode:** Stripe provides separate keys for test mode and live mode. Ensure you use the correct key depending on whether you are in the development or production phase.\n- **Security:** Keep your Secret Key secure and do not share it publicly or include it in client-side code. It provides full access to your Stripe account.\n\nFor more detailed information, refer to Stripe's official documentation on <a href=\"https://stripe.com/docs/keys\" target=\"_blank\" rel=\"noopener\">API keys</a>.\n\n#### Details (Optional)\n\n- **Tags:** Add any tags for categorization.\n- **Description:** Provide a brief description of the integration.\n\n#### Test and Save Integration\n\n- Click `Test Connection` to ensure the provided details are correct and the connection is successful.\n- Once the connection test is successful, click `Set up source` to complete the configuration.\n\n#### Troubleshooting\n\nIf you encounter any issues during setup:\n\n- **Check API Key:** Ensure the provided API key is correct and has the necessary permissions.\n- **Verify Account Information:** Confirm the account details are correct.\n- **Network Issues:** Check for any network issues that might be affecting connectivity.\n\n#### Frequently Asked Questions (FAQs)\n\n**How do I find my Stripe API key?**\n\nYou can find your API key in the Stripe Dashboard under Developers > API keys. More details can be found in <a href=\"https://stripe.com/docs/keys\" target=\"_blank\" rel=\"noopener\">Stripe's official documentation</a>.\n\n**What should I do if my connection fails?**\n\n- Ensure your API key is correct.\n- Verify your account details and ensure there are no typos.\n- Check your network connection and firewall settings.\n\n**How often is the data synced?**\n\nThe data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.\n\n**Can I customize the data being synced?**\n\nYes, you can customize the data being synced by adjusting the settings during the setup process.\n\n#### Support\n\nIf you need further assistance, please contact Boltic support or visit our <a href=\"https://docs.boltic.io\" target=\"_blank\" rel=\"noopener\">Support Documentation</a>.\n"
