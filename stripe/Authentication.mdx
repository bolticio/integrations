This guide walks you through setting up the Stripe integration within Boltic.

#### Prerequisites

To set up the Stripe integration, you'll need:

- The Stripe account email address
- The Stripe API key (Secret Key)
- Admin access to the Stripe account

#### Setting Up the Stripe Integration in Boltic

#### Log In

Log into your Boltic account or navigate to the Boltic Open Source dashboard.

#### Navigate to Integrations from the Primary Menu

Click on `Sources` and then click on `+ Add New source`.

#### Select Source Type

On the Set up the source page, search for `Stripe` from the search bar and select it.

#### Configure Integration

- **Name:** Enter a name for the Stripe integration.
- **Email:** Enter the email address associated with your Stripe account.
- **API Key (Secret Key):** Enter your Stripe API key. The Stripe integration uses the Secret Key for authentication (You can get the access key from <a href="https://dashboard.stripe.com/" target="_blank" rel="noopener">Stripe dashboard</a>).

#### Obtaining Your Stripe API Key (Secret Key)

To integrate with Stripe, you'll need your Secret Key for authentication. Follow these steps to obtain it:

1. **Log in to Stripe:**
   - Go to the <a href="https://dashboard.stripe.com/" target="_blank" rel="noopener">Stripe dashboard</a> and log in with your credentials.

2. **Navigate to API Keys:**
   - Once logged in, navigate to the **Developers** section in the left-hand menu.

3. **View API Keys:**
   - In the Developers section, click on **API keys**. Here you will see your **Publishable Key** and **Secret Key**.

4. **Reveal Secret Key:**
   - By default, the Secret Key may be partially obscured. Click on the "Reveal test key" or "Reveal live key" button next to the Secret Key to see the full key.

5. **Copy the Secret Key:**
   - Copy the Secret Key and use it in your application for authentication.

#### Important Notes

- **Test Mode and Live Mode:** Stripe provides separate keys for test mode and live mode. Ensure you use the correct key depending on whether you are in the development or production phase.
- **Security:** Keep your Secret Key secure and do not share it publicly or include it in client-side code. It provides full access to your Stripe account.

For more detailed information, refer to Stripe's official documentation on <a href="https://stripe.com/docs/keys" target="_blank" rel="noopener">API keys</a>.

#### Details (Optional)

- **Tags:** Add any tags for categorization.
- **Description:** Provide a brief description of the integration.

#### Test and Save Integration

- Click `Test Connection` to ensure the provided details are correct and the connection is successful.
- Once the connection test is successful, click `Set up source` to complete the configuration.

#### Troubleshooting

If you encounter any issues during setup:

- **Check API Key:** Ensure the provided API key is correct and has the necessary permissions.
- **Verify Account Information:** Confirm the account details are correct.
- **Network Issues:** Check for any network issues that might be affecting connectivity.

#### Frequently Asked Questions (FAQs)

**How do I find my Stripe API key?**

You can find your API key in the Stripe Dashboard under Developers > API keys. More details can be found in <a href="https://stripe.com/docs/keys" target="_blank" rel="noopener">Stripe's official documentation</a>.

**What should I do if my connection fails?**

- Ensure your API key is correct.
- Verify your account details and ensure there are no typos.
- Check your network connection and firewall settings.

**How often is the data synced?**

The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.

**Can I customize the data being synced?**

Yes, you can customize the data being synced by adjusting the settings during the setup process.
