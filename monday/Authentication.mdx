# Monday.com

This guide walks you through setting up the Monday.com integration within Boltic.

## Prerequisites

To set up the Monday.com integration, you'll need:

- A Monday.com account

If you don't have a Monday.com account, you can create one at <a href="https://auth.monday.com/users/sign_up_new?origin=hp_fullbg_page_header" target="_blank" rel="noopener">Monday.com</a>.

## Setting Up the Monday.com Integration in Boltic

1. **Create an API Token**
   - Log into your Monday.com account
   - Visit the <a href="https://boltic-company.monday.com/apps/manage?placement=dev_center" target="_blank" rel="noopener">Monday Developer Center</a>
   - Click on "My Access Tokens" in the left sidebar
   - Click "Generate New Token"
   - Copy and securely store your API token

2. **Log into Boltic**
   - Log into your Boltic account or navigate to the Boltic Open Source dashboard

3. **Navigate to Integrations**
   - Click on `Sources` in the navigation menu
   - Click on `+ Add New source`

4. **Select Source Type**
   - On the Set up the source page, search for `Monday` in the search bar
   - Select Monday.com from the results

5. **Configure Integration**
   - **Name:** Enter a descriptive name for your Monday.com integration
   - **API Token:** Paste the API token you generated earlier
   - **Workspace:** Select the workspace you want to connect

6. **Additional Details (Optional)**
   - **Tags:** Add relevant tags for better organization
   - **Description:** Provide a brief description of the integration's purpose

7. **Test and Save Integration**
   - Click `Test Connection` to verify your API token and settings
   - If successful, click `Set up source` to complete the configuration
