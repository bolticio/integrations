"# xAI Grok \n\nThis guide walks you through setting up the Grok integration within Boltic.\n\n## Prerequisites\n\n- xAI Grok API key.\n\n\n## Setting Up the Grok Integration in Boltic\n\n\n1.  **Log In**\n\n\t- Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n  \n\n2.  **Navigate to Integrations from the Primary Menu**\n\n\t- Click on `Sources` and then click on `+ Add New source`.\n\n  \n\n3.  **Select Source Type**\n\n\t- On the Set up the source page, search for `Grok` from the search bar and select it.\n\n  \n\n4.  **Configure Integration**\n\n\t-  **Name:** Enter a name for the Grok integration.\n\n\t-  **Authentication type :** Select API key as authentication type.\n\n\t-  **API_key:** Enter the API key associated with your Grok account.\n\n  \n\n5.  **Details (Optional)**\n\n\t-  **Tags:** Add any tags for categorization.\n\n\t-  **Description:** Provide a brief description of the integration.\n\n  \n\n6.  **Test and Save Integration**\n\n\t- Click `Test Connection` to ensure the provided details are correct and the connection is successful.\n\n\t- Once the connection test is successful, click `Set up source` to complete the configuration.\n\n\n## Troubleshooting\n\nIf you encounter any issues during setup:\n\n-  **Check Access Token:** Ensure the provided personal access token is correct and has the necessary permissions.\n\n-  **Network Issues:** Check for any network issues that might be affecting connectivity.\n\n\n\n## Overview\n\nThis page provides an overview of the xAI API, including authentication and example requests.\n\n## Authentication\n\nTo interact with xAI's API, you'll need to authenticate your requests. There are steps to follow:\n\nFirst, you will need to visit the xAI Console to create an API key. Later, you will need to include the API key in the \nAuthorization\n\n header of your requests. For more information, you can refer to our Quickstart <a href = \"https://docs.x.ai/docs/quickstart\" target=\"_blank\"> guide </a>.\n\n\n## Support\n\nIf you need further assistance, please contact Boltic support or visit our [ <a href = \"https://docs.boltic.io\" target=\"_blank\"> Support Documentation </a>]."
