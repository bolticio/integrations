# How to Get an Access Token for a Shopify Account

## Step 1: Create a Shopify Account & Store
1. Go to Shopify <a href="https://www.shopify.com/" target="_blank" rel="noopener"> <span> Support Documentation</span> </a>.
2. Click **Start free trial** and create an account.
3. Set up your **store name** (e.g., `mystore.myshopify.com`).
4. Complete store setup and go to your **Shopify Admin Dashboard**.

---

## Step 2: Enable Custom App Development
Before creating an app, you need to enable **custom app development** in your store.

1. In **Shopify Admin**, go to **Settings** > **Apps and sales channels**.
2. Click **Develop apps**.
3. If prompted, click **Allow custom app development**.
4. Click **Create an app**.

---

## Step 3: Create a New App
1. Enter a **name** for your app (e.g., "My API App").
2. Select the **App Developer** (your Shopify account).
3. Click **Create App**.

---

## Step 4: Configure API Access
To allow your app to access Shopify data:

1. Go to the **API Credentials** tab.
2. Click **Configure Admin API Scopes**.
3. Select the required **scopes** based on what your app needs:  
   - `read_products`, `write_products` (for product management)
   - `read_orders`, `write_orders` (for order access)
   - `read_customers`, `write_customers` (for customer data)  
4. Click **Save**.
5. Webhooks for required resource requires write scope.

---

## Step 5: Install the App & Get Access Token
1. Go to the **API Credentials** tab.
2. Click **Install App** (this generates an **Admin API Access Token**).
3. Copy the **Admin API Access Token** (it is shown **only once**).

## Step 6: Find Your Shopify Domain
To make API requests, you need your **Shopify store domain**.

1. Go to your **Shopify Admin Dashboard**.
2. Look at the browser URL:https://your-store-name.myshopify.com/admin
Your Shopify domain is your-store-name
