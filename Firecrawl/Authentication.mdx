# Firecrawl Authentication

## Overview

Firecrawl is a powerful web crawling and data extraction API that allows you to programmatically scrape, index, and retrieve content from websites at scale. It enables developers to automate data collection from the web efficiently using a simple, API-based interface.

## Authentication Ways

- API Key

## API Key

To authenticate with the Firecrawl API, you need an API Key generated from your Firecrawl account. This key must be included in your API requests to authorize actions and track usage.

**Steps:**

1. **Sign in to Firecrawl**  
   - Go to <a href="https://firecrawl.dev/" target="_blank" rel="noopener"><span>firecrawl.dev</span></a> and log in with your account credentials.

2. **Access API Settings**  
   - Navigate to your **Dashboard** or **API Settings** page.

3. **Generate or Copy Your API Key**  
   - Click **Generate** to create a new API key or **Copy** to retrieve an existing one
