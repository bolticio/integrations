"# Telegram Bot Authentication\n\nThis guide walks you through setting up Telegram Bot API authentication within the Boltic platform.\n\n## Prerequisites\n\n-   A **Telegram Bot Token**.\n\n## Setting Up Telegram Bot Authentication in Boltic\n\n1. **Create a Telegram Bot**\n\n    - Open the **Telegram app** on your device.\n    - Search for the **BotFather** bot (official Telegram bot creation tool).\n    - Start a chat with BotFather and use the command `/newbot` to create a new bot.\n    - Follow the prompts to choose a name and username for your bot.\n    - Once the bot is created, BotFather will provide you with a **Bot Token**. Copy this token, as you'll need it in the next steps.\n\n2. **Log In to Your Boltic Account**\n\n    - Log into your Boltic account or navigate to the Boltic Open Source dashboard.\n\n3. **Navigate to Integrations from the Primary Menu**\n\n    - Click on `Sources` and then click on `+ Add New source`.\n\n4. **Select Source Type**\n\n    - On the **Set up the source** page, search for `Telegram` in the search bar and select it.\n\n5. **Configure Integration**\n\n    - **Name:** Enter a name for the Telegram bot integration (e.g., \"Telegram Bot Integration\").\n    - **Authentication type:** Select **Bot Token** as the authentication method.\n    - **Bot Token:** Paste the **Bot Token** you received from BotFather into this field.\n\n6. **Details (Optional)**\n\n    - **Tags:** Add any tags for categorization (e.g., `telegram`, `bot`).\n    - **Description:** Optionally, provide a brief description of the integration (e.g., \"Telegram bot for receiving user messages\").\n\n7. **Test and Save Integration**\n\n    - Click **Test Connection** to ensure the provided details are correct and the connection to Telegram is successful.\n    - Once the connection test is successful, click **Set up source** to complete the configuration.\n\n## Troubleshooting\n\nIf you encounter any issues during setup:\n\n-   **Invalid Bot Token:** Ensure the token provided by BotFather is correct and has no extra spaces or characters.\n-   **Network Issues:** Check your internet connection to ensure there are no network issues blocking access to Telegram's servers.\n-   **Bot Permissions:** Ensure your bot has the necessary permissions to interact with Telegram APIs.\n\n---\n\n## Overview\n\nThis page provides an overview of the Telegram Bot API, including authentication and example requests.\n\n## Authentication\n\nTo interact with Telegram's API, you'll need to authenticate your requests using the **Bot Token** you received from BotFather.\n"
