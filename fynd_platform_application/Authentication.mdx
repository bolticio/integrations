# Fynd Commerce ( Application ) Integration

This guide walks you through setting up the Fynd Commerce ( Application ) integration within Boltic.

## Prerequisites

To retrieve application credentials, you'll need:

-   A registered <a href="https://platform.fynd.com/" target="_blank" rel="noopener"><span>Fynd Commerce account</span></a> with access to at least one sales channel in your company.

## Setting Up the Fynd Commerce ( Application ) Integration in Boltic

1. **Log In**

    - Log into your <a href="https://platform.fynd.com/" target="_blank" rel="noopener"><span>Fynd Commerce account</span></a>.

2. **Select Company**

    - If you have access to multiple companies, you will be prompted to select one. If not, the system will redirect you to the "Create Company" form.

3. **Access Fynd Commerce Dashboard**

    - Once you have selected the company, you will be directed to the Fynd Commerce Dashboard.

4. **Navigate to Developers Tab**

    - Click on the `Developers` tab in the sidebar. This action will open the `APIs & SDKs` page.

5. **Access Application Token**

    - Click on the `Application Token` tab to view a list of application credentials for all sales channels within the current company.

6. **Retrieve Credentials**

    - Locate the application credentials for your specific sales channel. The credentials include your `Application ID` and `Application Token`.

7. **Log into Boltic**

    - Log into your Boltic account or navigate to the Boltic Open Source dashboard.

8. **Navigate to Integrations**

    - Click on `Sources` and then click on `+ Add New source`.

9. **Select Source Type**

    - On the Set up the source page, search for `Fynd Commerce Application` from the search bar and select it.

10. **Configure Integration**

    - **Name:** Enter a name for the Fynd Commerce integration.

    - **Authentication Type:** Select Authentication Type for the integration.

    - **Application ID:** Enter the application ID obtained from the Fynd Commerce.

    - **Application Token:** Enter the application token obtained from the Fynd Commerce.

11. **Details (Optional)**

    - **Tags:** Add any tags for categorization.

    - **Description:** Provide a brief description of the integration.

12. **Test and Save Integration**

    - Click `Test & Save` to ensure the connection is successful.

    - Once the connection test is successful, your integration will be saved.
