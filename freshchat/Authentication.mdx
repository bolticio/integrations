

# Freshchat



This guide walks you through setting up the Freshchat integration within Boltic.



## Prerequisites

To set up the Freshchat integration, you'll need:

- A Freshchat account  
  You can create an account at <a href="https://www.freshworks.com/live-chat-software/" target="_blank" rel="noopener"><span>Freshchat Create Account</span></a>.



## Setting Up the Freshchat Integration in Boltic

1. **Log In**

   - Log into your Freshchat account.



2. **Obtain API Token**

   - Navigate to **Settings** > **API Tokens** > **Generate My API Token**.

   - Copy the API token generated and store it in a safe place.



3. **Log into Boltic**

   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.



4. **Navigate to Integrations**

   - Click on `Sources` and then click on `+ Add New source`.



5. **Select Source Type**

   - On the Set up the source page, search for `Freshchat` from the search bar and select it.



6. **Configure Integration**

   - **Name:** Enter a name for the Freshchat integration.

   - **API Key:** Paste the API token copied from the Freshchat settings.

   - **Region:** Select the applicable region from the dropdown menu.



7. **Details (Optional)**

   - **Tags:** Add any tags for categorization.

   - **Description:** Provide a brief description of the integration.



8. **Test and Save Integration**

   - Click `Test Connection` to ensure the provided details are correct and the connection is successful.

   - Once the connection test is successful, click `Set up source` to complete the configuration.



## Frequently Asked Questions (FAQs)

1. **How do I find my Freshchat API token?**

   - Log in to your Freshchat account and navigate to the Settings > API Tokens > Generate My API Token. Follow the instructions provided to generate and copy your API token.



2. **What should I do if my connection fails?**

   - Ensure the API token is correct.

   - Verify that the selected region corresponds to your Freshchat account's region.

   - Double-check the validity of the token and ensure it has not expired or been revoked.

   - Check your network connection and firewall settings for any potential issues.



3. **How often is the data synced?**

   - The data sync frequency is determined by the settings configured during setup. You can customize the sync frequency as needed.



4. **Can I customize the data being synced?**

   - Yes, you can customize the data being synced by selecting specific modules and adjusting the settings during the setup process. This allows you to tailor the data flow to suit your specific requirements.
