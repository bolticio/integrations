# Freshchat 

Use the Freshchat activity to automate work in Freshchat and integrate Freshchat with other activities. Freshchat provides a range of features, including creating, updating, deleting, and retrieving conversations, agents, reports, users, channels, and groups.

## Freshchat Secrets

### Prerequisites

- **Create a Freshchat account** - sign up at [freshworks.com/live-chat-software](https://www.freshworks.com/live-chat-software/).

### Supported Authentication Methods

- **API key**

### Related Resources

Refer to the [Freshchat API documentation](https://developers.freshworks.com/chat/api/) for more information about the service.

### Using API Key

To configure this secret, you'll need:

1. **API Key**: Refer to the [Freshchat API authentication documentation](https://developers.freshworks.com/chat/api/#authentication) for detailed instructions on obtaining your API key.
2. **Your Freshchat Domain**: Use the subdomain of your Freshchat account. This is part of the URL, for example, `https://<subdomain>.myfreshworks.com`. If you access Freshchat through `https://example.myfreshworks.com`, enter `example` as your Domain.

## Basic Operations

### Conversation
- Get a Conversation
- Create a Conversation
- Send a Message to a Conversation
- Update a Conversation Status
- List Messages in a Conversation

### Agent
- Get an Agent

### User
- Create a User
- Get a User
- Update a User
