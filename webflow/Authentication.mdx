

# Webflow

This guide walks you through setting up the Webflow integration within Boltic.



## Prerequisites

To set up the Webflow integration, you'll need:

- A Webflow account  
  You can create an account at <a href="https://webflow.com" target="_blank" rel="noopener"><span>Webflow</span></a>.



## Setting Up the Webflow Integration in Boltic

1. **Log In**

   - Log into your Webflow account.



2. **Log into Boltic**

   - Log into your Boltic account or navigate to the Boltic Open Source dashboard.



3. **Navigate to Integrations**

   - Click on `Sources` and then click on `+ Add New source`.



4. **Select Source Type**

   - On the Set up the source page, search for `Webflow` from the search bar and select it.



5. **Create a Connection**

   - Click `Create a Connection`.

   - Optional: In the **Connection name** field, enter a name for the connection.

   - Optional: Click `Show Advanced Settings` and enter your custom app's client credentials. To create a custom app, refer to the [Webflow OAuth2.0: App Registration documentation](https://university.webflow.com/lesson/oauth-2-0-authentication).

     - Note: If requested, use the following redirect URI: <a href="https://www.integromat.com/oauth/cb/webflow2" target="_blank" rel="noopener"><span>https://www.integromat.com/oauth/cb/webflow2</span></a>.



6. **Authenticate Webflow Account**

   - Click `Save`.

   - If prompted, log in to your Webflow account and confirm the access.



7. **Details (Optional)**

   - **Tags:** Add any tags for categorization.

   - **Description:** Provide a brief description of the integration.



8. **Test and Save Integration**

   - Click `Test Connection` to ensure the connection is successful.

   - Once the connection test is successful, click `Set up source` to complete the configuration.



## Frequently Asked Questions (FAQs)

1.  **How do I find my Webflow API token?**

- To obtain your Webflow API token, navigate to your Webflow project settings and look for the API tokens section. For detailed instructions, refer to Webflow's official documentation on [API tokens](https://university.webflow.com/lesson/api-tokens).



2.  **What should I do if my connection fails?**

- Ensure your API token is correct and valid.

- Verify that the client ID and client secret (if using a custom app) are accurate.

- Double-check the domain URL for any typos.

- Ensure your Webflow account has the necessary permissions.

- Check your network connection and firewall settings for potential issues.



3.  **How often is the data synced?**


- The data sync frequency depends on the settings you configure during the setup process. You can customize the sync frequency based on your needs.



4.  **Can I customize the data being synced?**

- Yes, you can customize the data being synced by selecting specific modules and configuring the relevant settings during the setup process.
